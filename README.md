
# TodoApp
>  To-do list iOS app

The purpose of this project is to show the basic functions of the following topics, and I hope you find it useful.

- CoreData.
- NSFetchedResultsController
- NSPredicate
- Change UI colors using Chameleon Framework
- SwipeCellKit
- SearchBar
- Segues
- MVC structure
